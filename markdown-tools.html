Markdown工具

Markdown是一种轻量级脚本语言，易写易读，适合用来编写文档，并将其纳入版本管理系统。本篇列举一组Markdown相关工具备忘。

eclipse插件

Markdown Editor plugin for Eclipse：可以利用eclipse中Help->Install New Software通过

http://winterstein.me.uk/projects/tt-update-site/ 安装。

提供Markdown标记高亮、预览和HTML导出等基础功能。

解析

各种语言都有对Markdown的解析库，比如Python，提供命令行的解析工具和模块调用两种方式解析，一般是转为HTML。如果希望转成其他格式的文档，可以利用Pandoc这个跨平台文档转换工具。

另外还找到一个页面，利用showdown.js（JS解析器）和一系列github的CSS，提供在线github的markdown显示风格预览，看此页代码，几乎所有的资源都是外链的，自定义代码就是通过JQuery调用，很简单。自己做一个用来展示文档也不错。

周边

  1. 插入说明类的图片（如流程图，架构图）：利用docs.google.com中的绘图画好流程图，然后文件->发布到网络，确定发布后，会得到图片外链URL，使用此URL到Markdown中即可。（注意：需要将URL中参数中;amp的变为&）

  2. 优势：不需要专门的图片托管/修改绘图后可以直接反映到文档中

  3. 劣势：访问google docs需要梯子

  4. 插入公式：利用在线Latex公式编辑器编辑公式，满意后选择下面输出为编码后的URL（url encoded）写入Markdown文档即可。

当然，上面两项不限于使用在Markdown文档编写。
